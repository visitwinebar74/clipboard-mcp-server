# 0.1.2 — 2026-05-25

## Fixed

- **`clipboard_read` RTF format** — unwrap the JXA-bridged string before testing truthiness; ObjC.unwrap(nil) is undefined (falsy), but the bridged wrapper object is truthy, causing a TypeError instead of `format_unavailable` when `public.rtf` is absent ([#1](https://github.com/cyanheads/clipboard-mcp-server/issues/1))
- **`clipboard_read` text format on empty clipboard** — `pbpaste` returns `""` on an empty clipboard, indistinguishable from a real empty string; now pre-inspects before calling `pbpaste` and throws `format_unavailable` when the text type is absent ([#2](https://github.com/cyanheads/clipboard-mcp-server/issues/2))
- **`stripHtmlTags` HTML plain-text fallback** — strip `<script>` and `<style>` blocks entirely (content and tags); insert a space before closing block-level tags (`p`, `h1–h6`, `div`, `li`, `td`, `tr`, `blockquote`, `pre`, and HTML5 semantic elements) and `<br>` so adjacent blocks don't merge after tag removal ([#3](https://github.com/cyanheads/clipboard-mcp-server/issues/3))
