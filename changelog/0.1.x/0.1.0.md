# 0.1.0 — 2026-05-25

Cross-platform clipboard MCP server with platform-detecting backend selection at startup.

## Added

- `clipboard_read` tool — read clipboard contents in text, HTML, RTF, or image format, with auto-select (richest format) mode; size-guarded (512 KB text/HTML/RTF, 5 MB image)
- `clipboard_write` tool — write plain text or HTML to the clipboard with automatic plain-text fallback for HTML; size-guarded (1 MB)
- `clipboard_inspect` tool — list available clipboard formats and byte sizes without reading full content
- macOS backend — `pbcopy`/`pbpaste` for text/HTML/RTF, `osascript` for image reads with PNG dimensions
- Linux X11 backend — `xclip` with TARGETS introspection
- Linux Wayland backend — `wl-clipboard` (`wl-paste`/`wl-copy`) with `--list-types` introspection
- Windows backend — PowerShell 5.1+ via `Get-Clipboard`/`Set-Clipboard` with `[System.Windows.Forms.Clipboard]` for types and images
- Platform detection at startup with actionable `ServiceUnavailable` errors when required CLI tools are missing
- Typed error contracts on all tools: `format_unavailable`, `content_too_large`, `clipboard_unavailable`
