# 0.1.4 — 2026-05-26

## Changed

- **`package.json` scripts** — migrated all script invocations from `tsx scripts/*.ts` to `bun run scripts/*.ts`; `start:stdio` and `start:http` now use `bun ./dist/index.js` instead of `node`; `test` now uses `bunx vitest run`; removed unused bare `start` script
- **`package.json` author** — changed from object form to canonical string `cyanheads <casey@caseyjhand.com> (...)`
- **`package.json` funding** — added GitHub Sponsors and Buy Me a Coffee funding entries
- **`package.json` repository** — added `git+` prefix and `.git` suffix to URL
- **`package.json` packageManager** — added `bun@1.3.2`
- **`package.json` files** — removed non-existent `AGENTS.md` and `Dockerfile` entries
- **`package.json` devDependencies** — removed `tsx` (no longer needed with `bun run` scripts)
- **README.md** — added npm badge; corrected Bun badge version to `v1.3.2`
- **`.github/FUNDING.yml`** — added with `github: cyanheads` and `buy_me_a_coffee: cyanheads`
- **`CLAUDE.md` commands** — updated command table from `npm run` / `tsx` to `bun run` pattern
